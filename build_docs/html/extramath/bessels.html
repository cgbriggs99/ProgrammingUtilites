Bessel Functions¶

These are Bessel and related functions.

Bessel Functions¶

These are the solutions to the Bessel differential equation,

\[y'' z^2 + y' z + \left(z^2 - \nu^2\right)y = 0\]

Complex Only¶

These functions have real counterparts defined in the C standard, but their complex versions were left out.

double _Complex cjn(int n, double _Complex x)¶
float _Complex cjnf(int n, float _Complex x)¶
long double _Complex cjnl(int n, long double _Complex x)¶
Parameters:

  • n – The parameter of the Bessel function.

  • x – The argument of the Bessel function.

Returns:

The value of the Bessel function of the first kind.

Finds the value of \(J_n(x)\). If x is zero, then the value returned is zero. Otherwise, the following formula is used.

\[\frac{x^n}{2^n n!} \sum_{k = 0}^\infty \frac{(-1)^k z^{2k}}{4^k (n + 1)_k k!}\]

This was chosen over the standard definition due to the \(4^k\) in the denominator, which should improve convergence.

long double _Complex cynl(int n, long double _Complex x)¶
Parameters:

  • n – The parameter of the Bessel function.

  • x – The argument of the Bessel function.

Returns:

The value of the Bessel function of the second kind.

Finds the value of \(Y_n(x)\). If x is zero, then the value returned is NAN to represent complex infinity. This is computed using a limit definition, since there does not seem to be a very good way to represent this function as a sum.

Real and Complex¶

These are definitions of functions for real and complex arguments.

double jnu(double nu, double x)¶
float jnuf(float nu, float x)¶
long double jnul(long double nu, long double x)¶
double _Complex cjnu(double _Complex nu, double _Complex x)¶
float _Complex cjnuf(float _Complex nu, float _Complex x)¶
long double _Complex cjnul(long double _Complex nu, long double _Complex x)¶
Parameters:

  • nu – The parameter of the Bessel function.

  • x – The argument of the Bessel function.

Returns:

The value of the Bessel function of the first kind.

Finds the value of \(J_\nu(x)\). This is computed using the following sum.

\[\frac{x^\nu}{2^\nu \Gamma(\nu + 1)} \sum_{k = 0}^\infty \frac{(-1)^k z^{2k}}{4^k (\nu + 1)_k k!}\]

This was chosen over the standard definition due to the \(4^k\) in the denominator, which should improve convergence. In some cases, this sum is skipped. These are

\[J_\nu(0) = 0; \Re(\nu) > 0 \vee \nu \in \mathbb{Z}\]
\[J_\nu(0) = \infty; \Re(\nu) < 0 \wedge \nu \not\in \mathbb{Z}\]
\[J_\nu(0) = \mathrm{NaN}; \Re(\nu) = 0 \wedge \nu \ne 0\]

double _Complex cynu(double _Complex nu, double _Complex x)¶
float _Complex cynuf(float _Complex nu, float _Complex x)¶
long double _Complex cynul(long double _Complex nu, long double _Complex x)¶
Parameters:

  • nu – The parameter of the Bessel function.

  • x – The argument of the Bessel function.

Returns:

The value of the Bessel function of the second kind.

Finds the value of \(Y_\nu(x)\). If \(\nu\) is an integer, then this uses the definition of yn for the appropriate type. Otherwise, it uses the following formula.

\[Y_\nu(z) = \csc(\pi\nu) \left(\cos(\pi\nu) J_\nu(z) - J_{-\nu}(z)\right)\]

Modified Bessel Functions¶

These are the solutions to the modified Bessel differential equation,

\[y'' z^2 + y' z - \left(z^2 + \nu^2\right)y = 0\]
double i0(double x)¶
float i0f(float x)¶
long double i0l(long double x)¶
double _Complex ci0(double _Complex x)¶
float _Complex ci0f(float _Complex x)¶
long double _Complex ci0l(long double _Complex x)¶
Parameters:

  • x – The argument to the Bessel function.

Returns:

The value of the modified Bessel function of the first kind.

Calculates \(I_0(x)\). Uses the following sum.

\[I_0(x) = \sum_{k = 0}^\infty \frac{z^{2k}}{4^k (k!)^2}\]

This skips the sum when \(x = 0\), since the value is simply 0.

Returns:

The value of the modified Bessel function of the first kind.

Calculates \(I_1(x)\). Uses the following sum.

\[I_n(x) = \frac{z}{2} \sum_{k = 0}^\infty \frac{z^{2k}}{4^k (k + 1)! k!}\]

This skips the sum when \(x = 0\), since the value is simply 0.

The value of the modified Bessel function of the first kind.

Calculates \(I_\nu(x)\). Uses the following sum.

\[I_\nu(x) = \frac{z^\nu}{2^{\nu} \Gamma(\nu + 1)} \sum_{k = 0}^\infty \frac{z^{2k}}{4^k (\nu + 1)_k k!}\]

The function returns some special values without computing the sum. These are

\[I_\nu(0) = 0; \Re(\nu) > 0 \vee \nu \in \mathbb{Z}\]
\[I_\nu(0) = \infty; \Re(\nu) < 0 \wedge \nu \not\in \mathbb{Z}\]
\[I_\nu(0) = \mathrm{NaN}; \Re(\nu) = 0 \wedge \nu \ne 0\]

long double knl(int n, long double x)¶
double _Complex ckn(int n, double _Complex x)¶
float _Complex cknf(int n, float _Complex x)¶
long double _Complex cknl(int n, long double _Complex x)¶
param n:

The parameter of the Bessel function.

param x:

The argument of the Bessel function.

return:

The value of the modified Bessel function of the second kind.

Calculates \(K_n(x)\). Uses the following definition.

\[K_n(x) = -\frac{\pi}{2} i^n Y_n(ix) + (-1)^n \left(i\ln(ix) - \ln(x)\right) I_n(x)\]

For the real functions, if this ends up being an imaginary number, NAN is returned instead.

  • x – The argument of the Bessel function.

Returns:

The value of the modified Bessel function of the second kind.

Calculates \(K_\nu(x)\). Uses the following definition for non-integer \(\nu\). For integer \(\nu\), the appropriate kn() is called.

\[K_\nu(x) = \frac{\pi}{2}\csc(\pi\nu)\left(I_{-\nu}(x) - I_{\nu}(x)\right)\]
